# EVA Esport Arena

Application mobile premium pour gérer l'équipe EVA Esport Arena.

## Objectif
Construire un hub interne pour l'équipe avec :
- scrims
- planning
- divisions
- notifications
- gestion des joueurs

## Stack prévue
- Expo
- React Native
- TypeScript
- Expo Router
- Firebase

## Direction artistique
- Noir profond
- Or métallique
- Marbre blanc / noir
- Interface premium et minimaliste

## Sprint 1
- Base Expo + TypeScript
- Design system EVA
- Navigation
- Écran d'accueil
- Dashboard de départ

## Lancer le projet
```bash
npm install
npm run start
```
